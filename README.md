## 📌 Descripción del proyecto
Este proyecto consiste en una aplicación web que implementa animaciones 2D mediante programación orientada a objetos.

Se desarrolla un sistema de partículas (círculos) que:

- Se generan desde distintas direcciones
- Simulan un comportamiento físico básico (gravedad)
- Rebotan al colisionar con los bordes
- Pierden energía progresivamente
- Desaparecen después de varios rebotes
- Se crean dinámicamente en intervalos aleatorios

---

## ⚙️ Tecnologías utilizadas
- HTML5  
- CSS3 (Glassmorphism + Bootstrap)  
- JavaScript (Canvas API)  
- Git y GitHub  

---

## 🚀 Funcionalidades principales
- Control del número máximo de elementos
- Ajuste dinámico del tamaño del canvas
- Selección de dirección de aparición
- Animación continua con `requestAnimationFrame`
- Generación automática (spawner)
- Simulación de gravedad y rebote
